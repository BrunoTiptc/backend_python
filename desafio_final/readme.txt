# 🏃 Projeto FastAPI – Desafio Final

Bem-vindo ao **Projeto FastAPI**, uma aplicação backend moderna desenvolvida como parte do meu portfólio de Engenharia de Software. Este projeto demonstra minhas habilidades em **desenvolvimento web, APIs RESTful e boas práticas de backend** usando Python e FastAPI.

---

## ⚡ Visão do Projeto

O objetivo deste projeto é criar uma API **rápida, escalável e intuitiva**, capaz de gerenciar dados de forma segura e eficiente. Este é o tipo de projeto que um **backend atleta** faria: ágil, preciso e pronto para qualquer desafio.

- **Backend moderno:** FastAPI
- **Banco de dados:** SQLite/MySQL/PostgreSQL (você escolhe)
- **Documentação automática:** Swagger UI / Redoc
- **Foco em testes e qualidade:** endpoints testáveis via PyTest ou Postman
- **Segurança:** validação de dados e tratamento de erros

---

## 🛠️ Ferramentas e Tecnologias

- **Python 3.11+**
- **FastAPI**
- **Pydantic** (validação de modelos de dados)
- **Uvicorn** (servidor ASGI)
- **SQLAlchemy** ou outro ORM de sua escolha
- **SQLite/MySQL/PostgreSQL**
- **PyTest** (testes unitários)
- **Postman** (testes manuais)
